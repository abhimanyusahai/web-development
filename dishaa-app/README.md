# Dishaa App

## Introduction

This is an app for improving and/or automating certain processes (like attendance tracking, assessment tracking etc.) at the educational organization "Dishaa" (based in Gurgaon).

## Getting Started

To get started with the app, first clone the repo into your local directory and then install the needed gems:

```
$ bundle install
```

Next, migrate the database:

```
$ rails db:migrate
```

Then run the test suite to ensure that everything is working correctly:

```
$ rails test
```

Finally, if the test suite passes, that means we are ready to run the app on a local server:

```
$ rails server
```
